# **App Name**: SUKHA OS

## Core Features:

- User & Access Management: Secure user authentication, role assignment with custom claims, and profile management to control system access.
- Guest & Reservation Lifecycle: Manage guest profiles, create new reservations, and handle existing bookings including modifications and cancellations.
- Check-in/Check-out Workflows: Streamlined process for guest check-ins (including walk-ins) and check-outs, updating room availability and initiating invoice generation.
- Room & Task Management: Real-time room status board, creation and assignment of housekeeping and maintenance tasks, and management of daily staff teams.
- Automated Billing & Invoicing: Automatic GST calculation and generation of detailed invoices for guest stays and other chargeable services.
- Real-time System Notifications: Display user-specific real-time notifications and alerts for critical system events or assigned tasks using Firestore listeners.

## Style Guidelines:

- Primary color: A deep, professional blue (#4D4DB3) that conveys reliability and calm, suitable for main interactive elements and branding.
- Background color: A very subtle light blue-gray (#F3F3F8) for clarity and reduced eye strain across data-rich interfaces.
- Accent color: A vibrant cyan-blue (#45C0E6) to highlight critical information, calls-to-action, and dynamic states.
- All text uses 'Inter', a grotesque sans-serif font known for its modern, neutral, and highly readable qualities, ideal for a functional management system.
- Utilize a consistent set of clean, vector-based icons that clearly represent common PMS functions such as check-in/out, room status, calendar, and user roles, maintaining a professional aesthetic.
- Implement a modular, card-based layout structure that clearly separates information sections, optimizes for desktop interaction, and prioritizes efficient data display and form interactions.
- Incorporate subtle and purposeful animations for state changes (e.g., button clicks, form submissions, real-time updates) to provide immediate user feedback without being distracting or slow.
